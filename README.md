# ZJY-Toolkit

> 🎓 智慧职教全功能辅助：24H云端签到监听、自动补签、出勤统计、课件秒刷、批量刷课、自动答题、试卷提取、答案查看、设备验证清除、多账号切换、邮件推送、CDK授权管理。

## 🌐 在线体验

**访问地址：** [https://study.atvkh.xyz](https://study.atvkh.xyz)

## ✨ 核心功能

### 📱 签到管理
- **24H云端挂机监听** — 服务器后台持续扫描签到活动，无需保持网页开启
- **自动补签** — 普通签到、手势签到、二维码签到一键云端补签
- **出勤统计** — 实时查看出勤数据与签到状态
- **代改考勤** — 支持代他人修改签到状态
- **邮件推送** — 签到结果实时邮件通知

### 📚 课程学习
- **课件秒刷** — 一键自动刷课，支持全部/只刷进度/只刷讨论
- **批量刷课** — 多课程并行处理
- **自动答题** — 自动搜索同学高分答案并提交
- **试卷提取** — 提取作业/考试试卷内容
- **答案查看** — 查看题库答案与历史成绩

### 🔧 考试辅助
- **设备验证清除** — 一键清除考试设备验证绑定
- **他人成绩查询** — 查看同课程其他学生成绩

### 👤 账号管理
- **多账号切换** — 无缝切换多个智慧职教账号
- **CDK授权管理** — 激活码生成、管理与授权体系
- **管理后台** — 用户管理、黑名单、系统监控

## 🚀 技术栈

- **后端：** Python + FastAPI 异步高性能框架
- **前端：** Vanilla JS 零依赖轻量化架构
- **部署：** 跨平台支持（Windows / Linux）
